About NLPCraft

NLPCraft is an open source library for building Natural Language Interface (NLI) for any type of applications.

Define a semantic model specifying how to interpret user input, how to query a data source, and how to format the results. Deploy this model and use REST API to securely explore the data source by using a free-form natural language from your applications.

Learn more

Why NLI

Natural Language Interface (NLI) enables users to explore any type of data source using a free-form natural language. NLI augments existing UI/UX by adding fidelity and simplicity of conversational AI.

Free-form natural language is the most intuitive and easiest way to interact with data. There is no learning curve, no special rules or applications to master, no syntax or terms to remember - just a natural language that your users already speak and the tools they already use.

Key Features
Semantic Modeling

NLPCraft enables advanced semantic modelling and comprehension allowing the free-form language understanding including anaphora and cataphora support, slang and spoken language comprehension.

Any Data Source

Any data source, device, or service - public or private. From databases and SaaS systems, to smart home devices and robots, to myriads of voice assistants and chatbots - NLPCraft can connect to any data source.

Java-First

While NLPCraft provides REST API for the application integration its data modelling APIs are Java/Scala-based natively supporting world's largest ecosystem of development tools, programming languages and services.

Strong Security

Model-As-A-Code

NLPCraft embraces 100% data model-as-a-code approach providing developers with native support for all the development, testing, devops, source code management and CI tools and frameworks existing in a vast Java eco-system.
