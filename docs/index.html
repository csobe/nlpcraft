About

NLPCraft is an open source library for building Natural Language Interface (NLI) into enterprise applications using voice or text. Developer defines a semantic model that specifies how to interpret user input, how to query or control a given data source, and how to format the final result back.

Once semantic model is defined and deployed the user applications use REST API to securely explore and control that data source by using a free-form natural language.

Learn more

Why NLI

Natural Language Interface (NLI) allows business users to explore and control public or private data sources using a free-form natural language. NLI augments existing UI/UX by adding fidelity and simplicity on conversational AI.

Free-form natural language is the most intuitive and easiest way to interact with data. There is a zero learning curve, no special rules, applications to learn, syntax, or terms to remember - just a natural language that your users already speak and the tools they already use.

Key Features
Semantic Modeling

NLPCraft enabled advanced semantic modelling and comprehension allowing for free-form language understanding including anaphora/cataphora support, slang and spoken language comprehension.

Any Data Source

Any data source, device, or service - public or private. From databases and SaaS systems, to smart home devices and robots, to myriads of voice assistants and chatbots - NLPCraft can connect to any data source.
